
# 🎵 Music Streaming App

A responsive music application that allows users to browse, search, and play songs with an elegant interface.

## Features

- Music Library
- Search Songs
- Audio Player
- Play / Pause
- Playlist Support
- Responsive Design

## Tech Stack

- React
- JavaScript
- CSS
- Node.js

## Installation

```bash
git clone <repository-url>

npm install

npm start
```

## Future Features

- User Accounts
- Favorites
- Lyrics Support
- Dark Mode
- Spotify API Integration
